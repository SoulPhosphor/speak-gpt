Phosphor Shines — Palette Designer

Design a theme by eye, then save it and hand the colors to Claude. Nothing here changes the app.
How to use this (click)
  • Dark / Light: you're designing two versions of each theme. Switch with the toggle — the pickers and preview follow the mode you're editing.
  • Calm / Vibrant: same colors, two treatments. Calm = solid single-color edges, no glow. Vibrant = multicolored gradient outlines + glow. Use it to design a sensory-safe version and a lively version of the same palette.
  • Reading surfaces are kept readable — message cards default to charcoal (dark) or off-white (light). Color lives in the edges, glow, accents and buttons. You can still change them, but that's the safe default.
  • Pick a color: click a swatch for the full-spectrum picker, or type/paste an exact hex.
  • Save it: name your theme and hit Save. It's stored in this browser. Use Copy for Claude to get a clean list to paste back.
  • Reduce motion previews the still version (no animation), for overstimulation-sensitive users.
  • Load "Legacy (current app)" only if you want to see the colors you're replacing. It is never the default.
